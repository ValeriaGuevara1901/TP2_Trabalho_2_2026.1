◆ BCD_ConverterLongASCII()

char * BCD_ConverterLongASCII ( int dimASCII,
char * pNumASCII,
long Numero )

Converte um inteiro long para string ASCII.

O string resultado estará alinhado à esquerda no buffer fornecido. Será truncado à direita caso o string convertido exceda a dimensão limite. O primeiro caractere será '-' se e somente se o número for negativo.

Parâmetros
dimASCIINúmero máximo de caracteres do string, incluindo o '\0' terminal.
pNumASCIIPonteiro para o buffer que receberá o string convertido.
NumeroInteiro a ser convertido para string.
Retorna
Retorna pNumASCII com o número convertido, ou NULL se pNumASCII==NULL ou dimASCII < 2.
Precondição
pNumASCII != NULL
dimensao(*pNumASCII) >= dimASCII
dimASCII >= max(3, 2 + log10(abs(Numero)))
